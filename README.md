# x402HQS — Safe Agentic Payments (x402 + IBA)

**Headquarters for x402 payments under Intent Bound Authorization governance.**

### Architecture
- **x402**: The payment rail (HTTP 402)
- **IBA**: The authorization gate (signed intent certificates + hard bounds + shard tokens)
- **WitnessBound**: The immutable audit record
